# FAQ commands — quick reference

These commands provide quick, consistent answers in Discord using embeds. Each command is server-only and replies ephemerally if used outside a guild.

Common behavior:
- Replies with a styled embed per topic
- Includes links, images, and step-by-step guidance when relevant
- Requires no options

Available commands:
- `/checkmate` — Map and comp board locations for the Checkmate contested zone.
- `/exechangar` — List of ships available from ExecHanger (with wiki links).
- `/fleetyard` — How to join and view the Even Legion Fleetyard.
- `/orbituary` — Map and comp board locations for the Orbituary contested zone.
- `/redkey` — Where to find and how to use the PYAM-Security KeyCard.
- `/ruin` — Map and comp board locations for the Ruin Station contested zone.
- `/uniform` — Legion uniform requirements and guidance.

Notes:
- All responses are embeds with static content; there are no command options.
- If you need content changes (text, links, or images), edit the corresponding file in `apps/arbiter/src/app/commands/faq/`.
